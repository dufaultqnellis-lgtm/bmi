# PlusFit - Healthy Lifestyle Platform

## 项目描述

PlusFit 是一个全面的健康生活平台网站，旨在帮助用户通过健身计划和营养指导改善生活质量。

## 功能特性

- **🏠 首页** - 平台简介和欢迎信息
- **💪 健身计划** - 初级、中级、高级三个锻炼等级
- **🍎 营养指导** - 饮食建议和营养知识
- **📊 BMI 计算器** - 完整的身体质量指数计算工具
  - 支持多种单位（厘米/米/英寸，千克/磅）
  - 自动分类（低体重/正常/超重/肥胖）
  - 个性化健康建议
- **📧 订阅功能** - 接收健康提示和更新

## 技术栈

- **HTML5** - 页面结构
- **CSS3** - 响应式设计和动画效果
- **JavaScript** - 交互功能和 BMI 计算

## 如何运行

### 本地运行

```bash
# 使用 Python
python -m http.server 8080

# 或使用 Node.js http-server
npx http-server -p 8080
```

然后在浏览器中访问：`http://localhost:8080`

## 功能说明

### BMI 计算器使用步骤

1. 输入身高（可选择单位：厘米、米、英寸）
2. 输入体重（可选择单位：千克、磅）
3. 点击"Calculate BMI"按钮
4. 查看结果和健康建议

### BMI 分类标准

- **低体重** - BMI < 18.5
- **正常体重** - BMI 18.5 - 24.9
- **超重** - BMI 25.0 - 29.9
- **肥胖** - BMI ≥ 30.0
